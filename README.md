# Project1_Quiz_game
#PROJECT 1 / 25 PYTHON 

Quiz Game (Python)

A simple command-line quiz game built using Python only for beginners.
The game asks general knowledge questions and calculates the final score based on correct and incorrect answers.

Features

Interactive command-line interface

Multiple-choice style questions (text input)

Case-insensitive answer checking

Scoring system:

+4 points for each correct answer

-1 point for each incorrect answer

Final score displayed at the end

How the Game Works

The program greets the user.

The user is asked whether they want to play.

If the user types yes, the quiz starts.

The quiz contains 4 questions related to:

Geography

General knowledge

German language basics

The score is updated after each question.

The final score is displayed once the quiz ends.

Requirements

Python 3.x

No external libraries are required.

How to Run

Clone the repository or download the file:

git clone <repository-url>


Navigate to the project folder:

cd quiz-game


Run the Python file:

python quiz_game.py

Example Scoring Logic
Correct Answer  → +4 points
Wrong Answer    → -1 point
