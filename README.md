# Quiz Game (Python)

A simple command-line quiz game built using Python.  
This project is beginner-friendly and focuses on basic Python concepts like input/output, conditions, and scoring logic.

---

## Features

- Interactive command-line quiz
- Case-insensitive answer checking
- Simple scoring system
- Final score displayed at the end

---

## How the Game Works

1. The program asks the user if they want to play.
2. If the user types `yes`, the quiz starts.
3. The quiz contains 4 questions.
4. Each correct answer gives **+4 points**.
5. Each wrong answer gives **-1 point**.
6. The final score is shown after all questions.

---

## Scoring System
Correct answer   → +4 points
Wrong answer     → -1 point
